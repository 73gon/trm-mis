### Caveat: `pp_AR` conflates model + decoder
`pp_AR` = `greedy_decoder(model_logits)`. The decoder picks highest-prob feasible nodes by structural priority, but the model contribution is **larger than initially estimated**. Back-filled (jobs 2504/2505):
- **SAT**: decoder-only `pp_AR = 0.6532`. SAT model lifts to 0.9983 → **+0.3451**, dominant signal.
- **ER-700**: decoder-only `pp_AR = 0.5949`. ER 2300's best 0.8849 lifts to **+0.2900** — the model **was** learning, contrary to the earlier read on bce/F1 alone. Trajectory variance came from greedy stochasticity on weak gradients, not zero signal.

→ Revised conclusion for ER 2300: real progress, but slow + interrupted by OOM. Phase B re-runs (B1 pw20, B2 deep) target faster convergence, not a "rescue from collapse".

→ All future runs log a decoder-only baseline (`--decoder_baseline degree`, on by default in `experiments/overfit_sl_pe.py`) and emit `model_lift = pp_AR − decoder_AR` to wandb (`eval_test/model_lift`) and to this table.
